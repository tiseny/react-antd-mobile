# react-antd-mobile
基于 antd mobile 的 react 项目模版

## 使用

**创建项目**

```bash
$ git clone https://github.com/tiseny/react-antd-mobile.git
$ cd react-antd-mobile  
```

**运行项目**

* 1. 本地开发运行

```bash
$ npm run dev
```

* 2. 构建

```bash
$ npm run build
```

* 3. devdll 本地静态化

```bash
$ npm run devdll
```

* 4. proddll 发布版本静态化

```bash
$ npm run proddll
```
